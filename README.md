# ⚡ Engro Energy: Thar Block II Command Center

**Strategic ESG, Grid Telemetry & Infrastructure Deployment Dashboard**

### 📌 Overview
This project is an enterprise-grade spatial strategy engine engineered specifically for operations in Tharparkar (Block II). It demonstrates how data analytics and geospatial mapping can optimize Corporate Social Responsibility (CSR) deployment and monitor critical energy infrastructure.

### 🚀 Key Features
* **Spatial Infrastructure Optimization:** Maps ideal deployment zones for Reverse Osmosis (RO) plants and Solar Microgrids based on localized water scarcity indices and population density.
* **CAPEX & ESG Modeling:** Features dynamic financial forecasting to allocate simulated budgets, automatically calculating community impact versus capital expenditure.
* **Grid Telemetry Simulation:** Monitors simulated megawatt loads for local substations to prevent grid overloads.
* **Fleet Safety Hub:** A dedicated dispatch interface simulating emergency monitoring for local fleet operations.

### 💻 Technical Architecture
* **Frontend:** Streamlit (Python)
* **Geospatial Mapping:** Folium, Plotly Express
* **Data Processing:** Pandas, NumPy
